Welcome to KOTM

What is this/How to use

This is a message board with a few big differences. There can only be one message at a time viewable, and everyone is allowed to fight for that spot. Enter your message into the box on the homepage and it will be on the site for all to see until someone else eventually replaces it. Everyone can enter their message as much as they like whenever they like, there is no time limit or anything like that. Anything besides html tags are welcome in the message no matter what it might say. If you want to see a list of every message ever posted you can see the log by clicking here.
